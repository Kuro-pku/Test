# Carrots — 塔防游戏（胡萝卜保卫战）

基于 **Qt 6 Graphics View Framework** 的 PvZ 风格塔防游戏。

---

## 已实现功能

### 核心玩法
- **地图**：13×7 地面网格，固定道路从左侧出怪点延伸至右侧胡萝卜
- **胡萝卜**：10 HP，3 档视觉状态（满血 / 半血 / 残血），血量归零隐藏，低于 10 显示爱心数字
- **波次系统**：从 `waves.txt` 加载，每波间隔 5 秒，最后一波全清立即胜利
- **金币**：初始 500，击杀怪物 +14，用于购买/升级炮塔

### 炮塔系统
- **放置**：顶部 PvZ 风格选择栏，点击选中 → 点击有效地面放置，金币不足灰显
- **升级**：3 个等级，攻击力 ×1.5、攻速 ×1.5、射程 ×1.2，等级以数字徽章显示（绿 1 → 橙 2 → 红 3）
- **攻击**：自动检测射程内最近怪物，发射豌豆子弹
- **朝向**：根据目标位置自动镜像 GIF（怪物在左 → 面朝左射击，在右 → 面朝右射击）
- **菜单**：点击炮塔弹出升级/删除图标，显示花费/返还金币，同时绘制攻击范围圈
- **删除**：返还总投入的 80%

### 怪物系统
- **怪物**：沿道路前进，抵达末端咬胡萝卜 1 滴血后消失
- **血条**：橙色圆角矩形，位于怪物上方
- **配置**：`towers.txt` 配置可用炮塔，`waves.txt` 配置波次内容

### 控制
- **暂停**：暂停/恢复游戏，等待当前出怪完成后再暂停
- **二倍速**：游戏帧率、出怪间隔、波次间隔同步 ×2，怪物间距不变
- **倒计时**：321 开场倒计时 GIF，期间禁止所有操作

### 结算
- **胜利**：最后一波清空，展示胜利图片 + 角色 GIF + 重新开始/返回菜单按钮
- **失败**：胡萝卜死亡，展示残血萝卜 + 角色 GIF + 重新开始/返回菜单按钮
- **重开**：完整清理怪物、炮塔、子弹、地面状态，重置金币和波次，重新进入倒计时

## 如何导入

### 环境要求
- **Qt 6.x**（MinGW 64-bit 或 MSVC）
- 需勾选 **Widgets** 模块
- GIF 播放依赖 **Multimedia** 模块（Qt 安装时勾选）

### 导入步骤
1. 将整个 `carrots` 文件夹（**除 `build/` 外**）拷贝到目标电脑
2. 打开 Qt Creator → 文件 → Open File or Project…
3. 选中 `CMakeLists.txt`
4. 选择构建套件（Kit）→ Configure Project
5. `Ctrl+R` 运行

### 注意事项
- **不要拷贝 `build/` 文件夹**，包含机器相关缓存
- `CMakeLists.txt` 已兼容 Qt 5/6
- 如报错 `qmovie` 或 `multimedia` 相关，确认 Qt 安装了 Multimedia 模块
